CONCLUSIONES

- Los campos mas importantes al crear un package.json son nombre y autor,
   que son como un identificador unico.

- Al crear una carpeta y quere colocar npm install, es necesario iniciar el servidor con npm init.

- Otro de los paquetes intalados es npm install underscore,
  es una biblioteca para JavaScript que proporciona soporte
  para los sospechosos funcionales habituales (cada uno, mapa, reducir, filtrar ...) 
  sin extender ningún objeto JavaScript central.
- En package.json, se guardarán todas las bibliotecas e installaciones que le pongamos al proyecto, junto a sus versiones.

- Usamos el comando npm adduser para poder validar nuestras credenciales en el proyecto que tenemos.
